# 字体文件说明

这个目录包含用于SVG文本导出的示例字体文件。

## 支持的字体格式

- **TTF** (.ttf) - TrueType字体
- **OTF** (.otf) - OpenType字体  
- **WOFF** (.woff) - Web开放字体格式
- **WOFF2** (.woff2) - Web开放字体格式2

## 字体来源

示例字体来自以下开源字体：

1. **Noto Sans** - Google开发的字体，支持多种语言
2. **Source Sans Pro** - Adobe开发的字体
3. **Roboto** - Google Material Design字体

## 使用方法

1. 在SVG文本导出器中点击"上传字体文件"
2. 选择本目录中的任意字体文件
3. 字体将自动加载并应用到SVG文本中
4. 可以导出为PNG、JPG或SVG格式

## 注意事项

- 字体文件仅供演示使用
- 商业使用请确保有相应的字体许可证
- 建议使用WOFF2格式以获得最佳性能
